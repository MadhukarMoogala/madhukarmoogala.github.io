Overview

APSCMD is a sample command exposed by the MyPlugin AutoCAD / OEM add-in. It demonstrates:

  • Command registration via the [CommandMethod] attribute.
  • Context (F1) help association using the native acedSetFunHelp function.
  • Safe access to the active editor inside a managed .NET plugin.
Tip: You can tailor this page for internal or public GitHub Pages hosting. After publishing, adjust the URL passed to SetFunHelp in AcadCommands.cs.

Syntax

APSCMD

The command requires no arguments in this sample implementation.

Behavior

When run, the command currently writes a diagnostic line to the AutoCAD command line:

Help Test - command executed.

You can extend the command to create geometry, query user input, modify drawings, or run automated tasks.

Extending the Command

Edit MyFirstCommand() to add geometry creation:

// inside MyFirstCommand()
using (var tr = doc.Database.TransactionManager.StartTransaction())
{
    var bt = (BlockTable)tr.GetObject(doc.Database.BlockTableId, OpenMode.ForRead);
    var ms = (BlockTableRecord)tr.GetObject(bt[BlockTableRecord.ModelSpace], OpenMode.ForWrite);
    var cir = new Circle(new Point3d(0,0,0), Vector3d.ZAxis, 10);
    ms.AppendEntity(cir);
    tr.AddNewlyCreatedDBObject(cir, true);
    tr.Commit();
}

F1 Help Integration

The add-in calls the native function acedSetFunHelp via P/Invoke:

SetFunHelp("APSCMD", "https://yourname.github.io/myplugin/apscmd.html#top", "", 0);

To enable this page after hosting on GitHub Pages:

  1. Create a repository named myplugin (or your choice).
  2. Enable GitHub Pages (Settings ? Pages) using main branch /docs or root.
  3. Place this file at docs/apscmd.html (or apscmd.html at root).
  4. Update the URL constant or variable in CommandsRegistrar.
  5. Rebuild and reload your bundle.

Troubleshooting

  • F1 does nothing: Ensure acedSetFunHelp returned 0; check command line warnings.
  • Firewall / blocked content: Host a local HTML file and pass a file path instead of HTTPS.
  • OEM build differences: Confirm accore.dll is available; otherwise the P/Invoke will fail.
  • Encoding issues: Keep UTF-8 without BOM for maximum compatibility.
