Drafting Tools

AutoCAD plugin providing quick-access ribbon buttons for common drafting workflows.

Commands

Zoom Selection

Zooms the viewport to the current selection set.

(c:ZOOMSEL)
Tip: Pre-select objects before clicking the button for best results.

Isolate Layer

Isolates the layer of a picked object — all other layers are hidden.

(c:LAYISO)
Tip: Use LAYUNISO to restore all hidden layers.

Loading the Plugin

Load DraftingTools.lsp via APPLOAD or add it to your AutoCAD startup suite.

Loading the Ribbon

Use CUILOAD and browse to DraftingTools.cuix to add the Drafting Tools tab to your ribbon.
